## Spacing Scale

The brand uses a **Fibonacci-derived** progression (not a strict 4-point grid).
This is a documented exception to the template default, with rationale: the
Fibonacci sequence maps to the zoetrope/mathematical motif and is expressed
throughout the design tokens. All gap aliases map to recognizable UI intervals.

### Fibonacci raw scale

| Token | Value | Usage |
|-------|-------|-------|
| `--space-0` | 0 | No spacing |
| `--space-1` | 2px | Hairline gaps, icon inner padding |
| `--space-2` | 4px (`--gap-xs`) | Icon-text gaps, pill inner padding |
| `--space-3` | 8px (`--gap-sm`) | Compact element spacing, status dot gaps |
| `--space-4` | 13px (Fib) | Metric row internal padding |
| `--space-5` | 21px (Fib) | Card content gaps |
| `--space-6` | 34px (Fib) | Section-to-section rhythm |
| `--space-7` | 55px (Fib) | Page-level vertical breathing |
| `--space-8` | 89px (Fib) | Hero / poster section |

### Everyday gap aliases (use these in Tailwind utilities)

| Token | Value | Tailwind utility | Usage |
|-------|-------|-----------------|-------|
| `--gap-xs` / `gap-1` | 4px | `gap-1` | Icon-label gap, badge internal |
| `--gap-sm` / `gap-2` | 8px | `gap-2` | Compact element spacing |
| `--gap-md` / `gap-3` | 12px | `gap-3` | Card internal item gap |
| `--gap-lg` / `gap-4` | 16px | `gap-4` | Default element spacing, section label bottom margin |
| `--gap-xl` / `gap-6` | 24px | `gap-6` | Section padding, card grid gap |
| `--gap-2xl` / `gap-8` | 32px | `gap-8` | Layout gaps, page horizontal padding |
| `--gap-3xl` / `gap-12` | 48px | `gap-12` | Major section breaks, page top padding |

Exception: 44px touch targets on mobile BottomTab items (≥44px per WCAG 2.5.5,
enforced via `min-height: 48px` on `.zt-tabitem`).

Source: `docs/design-system/tokens/spacing.css`

## Typography

Three families. Role determines family; size determines hierarchy within family.

### Font families

| Role | Family | Token | Fallback |
|------|--------|-------|---------|
| Display / headings / large readouts | Space Grotesk | `--font-display` | ui-sans-serif, system-ui, sans-serif |
| Body / UI text / nav labels | Hanken Grotesk | `--font-text` | ui-sans-serif, system-ui, sans-serif |
| Data / eyebrows / timestamps / units | Space Mono | `--font-mono` | ui-monospace, SFMono-Regular, monospace |

### Type scale (declared values — use only these sizes)

| Role | Token | Size | Family | Weight | Line-height | Tracking | Usage |
|------|-------|------|--------|--------|-------------|---------|-------|
| Micro label | `--text-2xs` | 11px | Mono | 400 | — | `0.12em` caps | Tab captions, eyebrow sub-labels, StatusBadge glyphs |
| Meta / timestamp | `--text-xs` | 12px | Mono | 400 | — | `0.08em` | Breadcrumbs, timestamps, unit labels, source attribution |
| Secondary body | `--text-sm` | 14px | Text | 400–500 | 1.5 | normal | Card sub-descriptions, table cells, detail copy |
| Body | `--text-base` | 16px | Text | 400 | 1.5 | normal | Nav labels, primary body, metric names in rows |
| Lead body | `--text-md` | 18px | Text | 500 | 1.65 | normal | Page subtitle/sub copy |
| Card titles | `--text-lg` | 22px | Display | 500–600 | 1.18 | `-0.01em` | Category names, card headings, section label headings |
| Section heads | `--text-xl` | 28px | Display | 500 | 1.18 | `-0.015em` | Section headings |
| Page titles | `--text-2xl` | 36px | Display | 600 | 1.04–1.18 | `-0.02em` | `PageHeader` titles, metric names (detail view) |
| Data readout (large) | `--text-3xl` | 48px | Display | 500 | 1 | `-0.02em` | Cessation day count, big stat readouts (`.zt-readout`) |

**Weights in use:** 400 (regular body), 500 (headings/medium), 600 (semibold active/titles), 700 (bold mono values, data emphasis).

**Line heights:**
- `--leading-tight` 1.04 — display / hero
- `--leading-snug` 1.18 — headings
- `--leading-normal` 1.5 — body
- `--leading-relaxed` 1.65 — long-form / detail copy

**Tabular numerals:** `font-variant-numeric: tabular-nums` on every element where numbers update — metric values, readouts, correlation r-values, cessation day counts. Use `.zt-tnum` helper class or `font-feature-settings: 'tnum' 1, 'ss01' 1`.

**Eyebrow pattern:** Space Mono 11px, weight 400, ALL-CAPS, `letter-spacing: 0.12em`, color `--text-muted`. Apply `.zt-eyebrow` class. This is the ONLY context where all-caps is used.

**Readout pattern:** Space Grotesk, weight 500, `font-variant-numeric: tabular-nums`, `letter-spacing: --tracking-tight`, `line-height: 1`. Apply `.zt-readout` class.

Source: `docs/design-system/tokens/typography.css` + `base.css`

## Motion

Frame-by-frame feel: brisk, eased, never bouncy.

| Token | Value | Usage |
|-------|-------|-------|
| `--dur-fast` | 120ms | Hover color, focus ring |
| `--dur-base` | 200ms | Card lift, theme transition, PhaseBar fill |
| `--dur-slow` | 320ms | Panel opens, accordion |
| `--dur-ring` | 900ms | MetricRing sweep-on-mount |
| `--ease-out` | `cubic-bezier(0.22, 1, 0.36, 1)` | Most transitions |
| `--ease-in-out` | `cubic-bezier(0.65, 0, 0.35, 1)` | Panel slide |
| `--ease-frame` | `cubic-bezier(0.83, 0, 0.17, 1)` | Stepped/mechanical moments |

TrendChart line sweep: `.zt-draw` CSS keyframe — `stroke-dasharray: 1; animation: ztDraw 900ms var(--ease-out) both`. Rests fully drawn; animation only when `prefers-reduced-motion: no-preference`.

Respect `prefers-reduced-motion`: `base.css` global rule sets all durations to `0.01ms !important` under `(prefers-reduced-motion: reduce)`.

Source: `docs/design-system/tokens/spacing.css` + `Zoetrope Redesign.html` styles

## Copywriting Contract

Voice: calm, precise, quietly encouraging. Address user as "you". Sentence case everywhere readable. ALL-CAPS only for Space Mono micro-labels. Numbers lead (tabular, with unit). No emoji. At most one "!".

| Element | Copy |
|---------|------|
| Primary CTA — WHOOP import | Save to tracker |
| Primary CTA — parse action | Parse WHOOP data |
| Primary CTA — correlations | Open correlations |
| Primary CTA — compare protocol | Compare versions |
| Secondary CTA — discard import | Discard import |
| Hero sub | Your signals, one frame at a time. 9 categories, tracked. |
| Dashboard eyebrow | Today's frame · [date] |
| Flagged metrics tile | Need a look |
| Section labels (eyebrow) | ALL-CAPS mono: METRIC CATEGORIES · CESSATION PROTOCOL · TOP CORRELATIONS · GENETIC INSIGHTS · METRIC STATUS · RANGE STATUS · TREND OVER TIME · MEASUREMENTS · DETAILS · VERSION HISTORY · SUPPLEMENTS BY TIER · FAAH CESSATION TIMELINE · PHASES |
| Empty state — no data | Nothing logged yet. Your first frame starts when you begin. |
| Empty state — no correlations | No correlations tracked yet. Import your first dataset to see supplement–metric relationships. |
| Parse loading state | Parsing… |
| Parse success state | [N] metrics read from your frame |
| Error state — parse failure | Something went wrong reading your file. Check the format and try again. |
| Destructive — discard import | Discard import: Your parsed data will not be saved. |
| Destructive — remove file | Remove file: The uploaded file will be cleared. |

| Wordmark | `zoetrope.` — lowercase, period in periwinkle. Small `codename Zoetrop` tag alongside (flag only; do not relitigate naming). |
| Metric status words | optimal · borderline · deficient · excess (always lowercase in body; ALL-CAPS in mono badges) |

## Category System (Q1 + Q1b Resolution)

**Exception (Q1 family tint):** Autonomic and Body Composition have a `family` property in the data that activates a soft family tint on their CatChip. All other categories render neutral.

**Active chip state** (when category is selected in nav): background → `--ink`, color → `--n-50` (inverted), regardless of family.

```

The `color` field (currently Tailwind color class e.g. `text-amber-500`) is replaced by the family-driven logic above (no per-category color class).

## Dark Theme (Q3 Resolution)

**Light is primary.** The warm-dark theme is defined and token-driven, delivered by the roundtrip.

### Implementation

No-flash inline script in `<head>` (before first paint):
```js
(function () {
  try {
    var t = localStorage.getItem('zt-theme');
    if (t !== 'dark' && t !== 'light') {
      t = window.matchMedia && window.matchMedia('(prefers-color-scheme: dark)').matches ? 'dark' : 'light';
    }
    document.documentElement.setAttribute('data-theme', t);
  } catch (e) {
    document.documentElement.setAttribute('data-theme', 'light');
  }
})();
```

Theme applied via `html[data-theme="dark"]` CSS selector overriding the warm neutral ramp and metric tints.
