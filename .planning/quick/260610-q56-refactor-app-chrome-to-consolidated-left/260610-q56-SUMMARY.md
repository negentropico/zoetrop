# Quick Task 260610-q56: Consolidated Left Nav Summary

One consolidated left sidebar (264px expanded / 64px icon rail / off-canvas mobile drawer) replaces TopNav + BottomTab + 5 per-section sub-navs, with cookie-persisted collapse (SSR-read, no flash), shell-wide breadcrumbs from nav-tree.ts, and a flattened routes.ts that fixes the unroutable /ingest.
