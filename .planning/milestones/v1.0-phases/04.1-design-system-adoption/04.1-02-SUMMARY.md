## What Was Built

### app.css — Three-layer token bridge

1. **Layer 1 (tokens):** All DS token vars from `colors.css`, `typography.css`, `spacing.css` inlined directly — `@import` fallback applied because `docs/` is outside the Vite root and the path would escape the build.
2. **Layer 2 (additions):** `--excess: #c98910` / `--excess-bg` (D-06), round1 reduced radii (`--radius-xl: 14px`, D-12), semantic surface/border aliases.
3. **Layer 3 (`@theme inline`):** All color/font/radius/shadow tokens mapped to Tailwind utility namespaces using `var(--token)` — `inline` keyword is mandatory so utilities reference vars at runtime (dark remap cascades).
4. **Dark remap:** `html[data-theme="dark"]` block overrides the neutral ramp + ink + family washes + excess + shadows (from roundtrip).
5. **Scoped transitions:** Only on `html, body, .zt-topnav, .zt-bottomtab, .zt-theme-toggle` — global `* { @apply transition-colors }` anti-pattern removed.
6. **Base helpers:** `.zt-eyebrow`, `.zt-readout`, `.zt-tnum`, `.zt-draw` keyframe, `.zt-grid-2/3/4` + responsive reflow, `.zt-trow/.zt-mrow` hover, reduced-motion, focus-visible ring, `::selection`.

### Shell + Theming Primitives

- **SpiralMark:** Golden spiral SVG path data copied verbatim from `lib.jsx` (line 75). Props: `size`, `color` (default `var(--ink)`).
- **Wordmark:** react-router `Link to="/"`, SpiralMark + "zoetrop" + periwinkle "." + "codename Zoetrop" Space Mono tag (D-08).
- **ThemeToggle:** 40px button, Sun in dark / Moon in light (lucide-react), localStorage `zt-theme`, initial state from `documentElement.getAttribute('data-theme')` (no client flash), accessible aria-label.
- **TopNav:** Sticky, 5-item NAV constant, active-underline NavLink pattern, solid `var(--paper)` fallback before `color-mix()` backdrop (Pitfall 8/A6), Wordmark + ThemeToggle, `hidden md:flex` desktop-only link cluster.
- **BottomTab:** Fixed `md:hidden`, 5 Lucide icon+label items, `min-height: 48px` per WCAG 2.5.5 (UI-01-l), active: `var(--accent)` / inactive: `var(--text-muted)`, Space Mono 10px uppercase labels.
- **AppShell:** TopNav + `<main>` + footer (SpiralMark) + BottomTab + 80px mobile spacer.

### root.tsx Rewire

- Inter replaced with Space Grotesk 400/500/600/700 + Hanken Grotesk 300–800 + Space Mono 400/700 via Google Fonts.
- `suppressHydrationWarning` on `<html>` silences SSR data-theme mismatch.
- No-flash script (`NO_FLASH_SCRIPT` static literal) injected via `dangerouslySetInnerHTML` before `<Meta />/<Links />`.
- Body: `bg-paper text-ink min-h-screen font-text` (old `bg-gray-50 dark:*` removed).
- `App()` renders `<AppShell><Outlet/></AppShell>`. Inline `Header` function deleted.

## Deviations from Plan

### Auto-fixed Issues

**1. [Rule 3 - Blocking] reactRouter() Vite plugin crashes jsdom tests**
- **Found during:** Task 2 ThemeToggle test run
- **Issue:** `npm test -- run ThemeToggle.test.tsx` failed with "React Router Vite plugin can't detect preamble" — the plugin intercepts all module transforms including test files
- **Fix:** Updated `vite.config.ts` to conditionally exclude `reactRouter()` when `process.env["VITEST"]` is set (standard workaround)
- **Files modified:** `remix-app/vite.config.ts`
- **Commit:** c0acee1

**2. [Rule 3 - Blocking] vitest 4 jsdom localStorage non-functional stub**
- **Found during:** Task 2 ThemeToggle test run (after fixing blocker 1)
- **Issue:** vitest 4.x passes `--localstorage-file` without a valid path, leaving `localStorage.removeItem` as a non-function in the jsdom environment
- **Fix:** Added `remix-app/app/test-setup.ts` with a Map-backed localStorage polyfill; referenced in `vite.config.ts` `test.setupFiles`
- **Files modified:** `remix-app/app/test-setup.ts`, `remix-app/vite.config.ts`
- **Commit:** c0acee1

**3. [Rule 1 - Bug] Testing Library duplicate renders across tests**
- **Found during:** Task 2 ThemeToggle test run (after fixing blockers 1 and 2)
- **Issue:** Tests 2–6 found "multiple elements with role button" because renders from prior tests accumulated in the DOM without cleanup
- **Fix:** Added `afterEach(() => cleanup())` import from `@testing-library/react` in `ThemeToggle.test.tsx`
- **Files modified:** `remix-app/app/components/ui/ThemeToggle.test.tsx`
- **Commit:** c0acee1

**4. [Rule 2 - Missing critical] TopNav needs solid background fallback before color-mix()**
- **Found during:** Task 2 implementation
- **Issue:** Plan specified `color-mix()` backdrop for glassmorphism but Pitfall 8/A6 requires a solid fallback for Safari <16.2
- **Fix:** Added `background: var(--paper)` as a separate style on the header, then applied `color-mix()` + `backdrop-filter` on a child overlay div
- **Files modified:** `remix-app/app/components/shell/TopNav.tsx`
- **Commit:** c0acee1

**5. [Rule 2 - Missing critical] A7 @import path fallback applied**
- **Found during:** Task 1 implementation (pre-build analysis)
- **Issue:** `@import "../../docs/design-system/tokens/colors.css"` from `remix-app/app/app.css` would resolve to `remix-app/docs/...` — the `docs/` directory is at the repo root, outside the Vite project root (`remix-app/`), which would cause a Vite build error
- **Fix:** Inlined all three token file contents directly into `app.css` (documented A7 fallback from PLAN.md)
