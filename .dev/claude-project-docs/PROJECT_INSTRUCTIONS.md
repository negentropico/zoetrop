## Critical Safety Rules (Memorize These)

### NEVER Recommend
- **GABAergics**: Valerian, hops, kava, benzos, Ambien, phenibut, passionflower (~50% ASD paradox risk)
- **Hepatotoxins**: Alcohol, acetaminophen, high-dose niacin (NAFLD 98th%)
- **Methyl donors**: SAMe, TMG, betaine, high-dose methylcobalamin (COMT intermediate)
- **Catecholamine precursors**: L-tyrosine, L-DOPA, Mucuna (COMT intermediate)
- **B-vitamin mega-doses**: AG1, B-complex, B6 >25mg (previous toxicity: B6 102.9 μg/L)
- **Serotonergics**: 5-HTP, tryptophan, St. John's Wort
- **High caffeine**: >200mg or after 10 AM (CYP1A2 slow metabolizer)
- **High-dose melatonin**: >0.5mg (ADHD - use 0.3mg chronobiotic only)

### Always Verify Before Recommending
- Any new supplement → check 07_Safety.md first
- Any sleep aid → confirm not GABAergic
- Any stimulant/adaptogen → consider COMT status
- Any B-vitamin → remind about previous toxicity, suggest blood work first
