# Zoetrop

Comprehensive wellness tracking dashboard consolidating WHOOP biometrics, blood work, body composition, and protocol progress.

> **Status:** v1.0 "M1 Foundations" shipped 2026-06-14 (auth + tenant/subject RLS, live Neon data layer, lab-ingest → report pipeline). v1.1 "First Client (practitioner-operated)" is now executing — a practitioner onboards one real client end-to-end. The direction — growing into a multi-tenant functional-health operations platform — is in [`docs/PLATFORM.md`](docs/PLATFORM.md). `Zoetrop` is the internal codename; the public brand is deferred (see [`docs/NAMING.md`](docs/NAMING.md)).

## Tech Stack

- **Framework**: React Router 7 (Remix)
- **Language**: TypeScript 5.x (strict mode)
- **UI**: React 19 + Tailwind CSS 4 + Recharts
- **Database**: Neon Postgres + Drizzle ORM
- **Deployment**: Vercel

## Quick Start

```bash
cd remix-app
npm install
npm run dev        # Start dev server
npm run build      # Production build
npm run typecheck  # Type generation + type check
```

## Project Structure

```
remix-app/
├── app/
│   ├── components/    # React components (TrendChart, nav, metrics)
│   ├── lib/           # Real data modules (blood work, WHOOP, protocols)
│   ├── routes/        # Explicit routing via `app/routes.ts` (RouteConfig)
│   └── types/         # TypeScript interfaces (metrics, protocol)
├── db/
│   └── schema.ts      # Drizzle table definitions (636 lines, 22 tables)
├── drizzle.config.ts  # Drizzle ORM configuration
└── package.json
```

## Routes

The authenticated app (everything under `routes/_app/`) is gated by Better-Auth —
unauthenticated requests get a 302 to `/login`.

| Route | Description |
|-------|-------------|
| `/` | Landing page (public) |
| `/login`, `/logout` | Auth (public) |
| `/dashboard` | Dashboard with cessation tracker, metric stats (auth) |
| `/metrics/:category` | Category detail views (auth) |
| `/metrics/:category/:metricId` | Individual metric detail (auth) |
| `/protocol/versions` | Protocol version history (auth) |
| `/protocol/supplements` | Supplement tiers (auth) |
| `/protocol/cessation` | Cessation timeline (auth) |
| `/protocol/compare` | Version comparison (auth) |
| `/insights/correlations` | Metric correlations (auth) |
| `/insights/genetics` | Genetic variants (auth) |
| `/import/whoop` | WHOOP data import (auth) |
| `/import/vault` | Obsidian vault import (auth) |
| `/ingest`, `/ingest/upload`, `/ingest/review` | Lab-ingest pipeline (auth) |
| `/reports`, `/reports/generate` | Confidence-graded report generation (auth) |
| `/clients`, `/clients/new` | Practitioner client management (auth) |
| `/settings`, `/settings/assignments` | Account + subject assignments (auth) |

## 9 Metric Categories

1. **Vitamins** - B-vitamins, fat-soluble
2. **Minerals** - Zn, Mg, Fe, trace
3. **Inflammatory** - hs-CRP, homocysteine
4. **Metabolic** - Glucose, kidney
5. **Hormones** - Sex, thyroid, cortisol
6. **Autonomic** - HRV, RHR, sleep (WHOOP)
7. **Body Composition** - DEXA, lean mass
8. **Lipids** - Cholesterol, triglycerides
9. **Hematology** - CBC, WBC
