Icon-only button for toolbars, nav and card headers — wraps a single icon node passed as children.

```jsx
<IconButton label="More" variant="ghost"><i data-lucide="ellipsis"></i></IconButton>
<IconButton label="Add" variant="ink" round><i data-lucide="plus"></i></IconButton>
```

Variants `ghost`/`soft`/`outline`/`ink`; sizes `sm`/`md`/`lg`. Set `round` for a circle. Always pass `label`.
