# Zoetrop — Naming Exploration (paused 2026-05-20)

Record of the venture-brand naming hunt for this platform, so it can be resumed without
re-running the ~45 collision checks already done. **Current state: public brand deferred;
`Zoetrop` formalized as the internal codename** (matches `zoetrop.netlify.app`), replacing
the old "Tracker" name.

> Methodology mirrors the Trouvant brand convention (`Dutch/plan/phase1-research/brand/BRAND_NAMING.md`):
> 2–3 syllables · soft close · rewards definition · register-fit · **in-vertical collision check is a
> blocking gate** (phonetic adjacency to a health-vertical competitor blocks, per the Reprise→Renvoi
> precedent). Real USPTO Class 42/44/41 + domain + handle checks are a later step, not done here.

---

## What we're naming

The platform formerly codenamed **Tracker**: a pharmacogenomic wellness dashboard (personal n=1
instrument) growing into a **B2B white-label, multi-tenant PaaS for functional-health practitioners**
(coach / nutritionist / trainer; group + 1:1; branded client app; programming, monitoring, marketing,
retreats). NGT venture seed. **Not a B2C brand** — it must read credible to *practitioners* (Tara-profile),
and stays invisible behind their brand.

## Register (locked through iteration)

1. ❌ **Precious-literary** (French/Greek etymological, Trouvant-style) — rejected outright ("yuck").
2. ✅ **Reference brands Mac likes: ZOE and Function Health.** ZOE = a biology word ("life") as a clean
   modern brand. Function = **in-domain** — "functional health" + "the ability to function well"
   (NOT an engineering-borrow). Target = confident, clean, science/contemplative, says-it-without-a-lecture.
3. ✅ **Final lean: spiritual / Eastern** (Buddhism, Sanskrit, Zen, Eastern traditions) — less Greek/Latin.
   Fits the product (meditation, breathwork, mindfulness, life-coaching, retreats) and Tara's HIGHER world.
4. **The "path / guide" angle is the keeper** (Marga landed conceptually; the word itself was killed).

## Structural finding (the important one)

The functional-health / coaching / wellness-SaaS naming space is **one of the most saturated in tech.**
~45 candidates across 6 rounds — nearly every common dictionary word (English health/whole/function/vital,
Greek/Latin roots, *and* the Sanskrit/Pali yoga vocabulary) is already taken in-vertical, many dead-on.
**Ownership only lives in:** coined words (ZOE, Healthie, Noom, Oura), short *uncommon* roots, or two-word
lockups (Function Health, Practice Better, Heads Up Health).

## Hard constraints (do not relitigate)

- **Avoid MAGA-adjacency** — killed both `Magga` and `Marga`.
- **Any "Zoe-" name is dead** — ZOE (Tim Spector) is the category leader in personalized-nutrition-from-biology.
  (The codename `Zoetrop` keeps the *tropos*/"turning" half of *zoetrope*, not the *zoe* half — and is
  internal-only, so the collision doesn't bite.)
- Buyer = functional-health practitioners; register = contemplative-premium, practitioner-credible.
